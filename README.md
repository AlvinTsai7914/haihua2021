# haihua2021
### 簡介
海外民俗文化種子教師線上培訓班網站與課程系統
配合客戶需求使用 bootstrap4、scss 修改課程卡片樣式與效果，以及整體顏色版面的修改，使用 bootstrap4 model 製作學員交作業系統和忘記密碼表單。
